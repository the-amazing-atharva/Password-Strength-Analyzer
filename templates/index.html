🔓 Breach Check (Have I Been Pwned)

This section checks whether your password has appeared in known data breaches using the Have I Been Pwned database. If a password is found, it means it has been exposed publicly and should never be used again.

The check uses a secure method (k-anonymity) to protect your password from being fully exposed.
